# Data Visualization: criando gráficos com bibliotecas Python

Este projeto aplica técnicas avançadas de **Data Visualization** e manipulação de dados para analisar o fluxo imigratório para o Canadá ao longo das últimas décadas. O foco principal é transformar dados brutos em insights visuais claros, utilizando o poder da biblioteca **Pandas** e estilização via **CSS**.

---

## Objetivos do Projeto

Desenvolvido como parte do curso de **Data Visualization**, este projeto cobre:

* **Manipulação de DataFrames**: Trabalho com dados reais para apresentar informações migratórias de forma estruturada.
* **Classe Styler do Pandas**: Uso profundo da biblioteca para estilizar tabelas e destacar tendências.
* **Built-in Styles**: Aplicação de estilos prontos do Pandas para facilitar a análise rápida (máximos, mínimos e gradientes).
* **Estilização Customizada com CSS**: Uso de propriedades CSS para criar tabelas profissionais, controlando fontes, cores, bordas e alinhamentos.
* **Visualização de Cenários**: Construção de tabelas e gráficos adaptados para diferentes contextos de análise de dados.

## Tecnologias e Bibliotecas

Para garantir a execução correta das estilizações e gráficos, o projeto utiliza:

- **Python 3.x**
- **Pandas**: Para análise e a classe `Styler`.
- **Jinja2**: Essencial para renderizar o CSS nas tabelas do Pandas.
- **Matplotlib**: Para criação de gráficos de séries temporais.
- **Seaborn**: Para aprimoramento visual das plotagens.
